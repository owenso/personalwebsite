While at General Assembly

During the 3 Months I was in the Web Development Immersive Bootcamp at General Assembly, we were given four projects to complete.

Blackjack

Our first assignment at General Assembly was to make a Blackjack app using Javascript, HTML, and CSS. I decided to go beyond that. I taught myself Firebase, then added online multiplayer functionality and chat.

Github Live Project

MakeMeA

Our second assignment at General Assembly was to make a Forum using Node.js and Express. I decided to incorporate WebRTC and create a webapp which allows users to make requests for creative content, and allows other users to fill those requests via webcam audio or video recording as well as image uploads.

Github Live Project

SOS NOW

Our third assignment at General Assembly was to make a Rails webapp. I worked as part of a three person team to create a post-disaster rescue app which would allow someone to submit information and their geolocated position to a database, to assist rescuers. The app also sends out a tweet with information.

Github Live Project Twitter

Workout Tracker

For my final assignment at General Assembly, I decided I wanted to learn the rest of the MEAN Stack. So I taught myself Mongo (with Mongoose) and Angular, and I built a simple app which allows a user to plan and track their workouts.

Github Live Project
